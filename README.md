# Hi, I’m Andrej Glavnik

I’m a Data Analytics Project Manager / Technical Project Manager focused on analytics operations, dashboard governance, tracking QA and cross-functional BI delivery.

I sit between business teams, stakeholders, vendors, analytics teams, developers and technical systems. My strongest work is making messy reporting and analytics delivery usable: requirements, KPI definitions, QA checks, dashboard logic, issue investigation, documentation and stakeholder alignment.

## How I Think

Good analytics delivery is not only about building a dashboard. It is about making sure the right metric is defined, the source is understood, the data is checked, the owner is clear, the stakeholder knows what changed and the next person can maintain it.

That is the operating layer I like to own.
